# Introduction to Mitsi

Mitsi a modern, open-source video conferencing platform powered by mediasoup sfu. Inspired by Jitsi, Mitsi delivers low-latency, scalable video and audio calls, screen sharing, and real-time chat using WebRTC. Built with a microservices architecture, Mitsi is designed for extensibility, ease of deployment, and community contributions.
